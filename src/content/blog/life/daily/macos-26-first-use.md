前不久抱着试试看的心态更新了 macOS 26。

从最初的惊艳到后来发现的各处细节问题，再到从各处角落发现一些不错的改进，感觉这个升级并不像很多人说的一样一无是处，它对于大多数用户其实还是一个不错的体验进化，而且进化了不少（前提是硬件带得动，但实际上只要是 Apple Silicon 现在应该都带得动了）。

于是写个文章来总结一下我发现的 macOS 26 的改进与问题，希望能给像我一样迟疑是否应该升级系统的伙伴一点参考。

:::info no-icon

本文的所有信息均基于写作时的 macOS Tahoe 最新正式版 26.3，如果后续系统更新中出现不一致的行为，本文大概率不会更正或补充，请自行注意甄别本文所含信息的时效性。

:::

## 启动台与全局搜索

### 启动台

这是这次更新中可能最影响体验的一项更改：启动台（Launchpad）被砍了！

如果你十分在意启动台，而对新版本系统的一些新特性（下面会讲）不在乎，千万不要更新，它就是直接 R.I.P. 了，而非隐藏起来或是给了切换选项。取而代之的，苹果整了个功能很简单的 Application 面板，简单到连文件夹都不支持。

![Application 面板](https://s3-blaze.rqwq.link/vscode/d7861985e5cd3a8939e999dde5c53cee.png)

面板上方有一些采用类似 iOS 侧边抽屉分类方式的选项，可以用于快速查找相应类别的应用，但我体验下来的感受就是——废物，是的没错就是废物，一点效率提升都没有。不过好在我已经习惯了 Spotlight 启动应用，所以这个更改对我影响不算大，最多有点不顺眼。

这个面板目前还有个容易触发强迫症问题，就是每次打开的时候都会加载一遍图标。它的缓存似乎没有很丝滑，曾经的启动台图标是几乎不会让用户看到加载过程的，但现在的效果就是，总有那么几次，打开会发现应用图标由一块透明区域稀里哗啦的冒出来，而不是本来就在那。

### 聚焦搜索（Spotlight）

苹果对 Spotlight 进行了一波改进，原先单调的一个框现在变成了一个框冒出来 4 个按钮。

为什么要强调冒出来呢？因为它就是冒出来的。

{% media video %}
- url: https://s3-blaze.rqwq.link/vscode/49d171914f39262574bec2fc0cb3b961.webm
{% endmedia %}

实际上就是给 Spotlight 加了个分类功能，同时让它能搜索更多东西了。现在可以在 Spotlight 中搜索本机文件、App Store、网络内容等等很多东西，这个改进还是比较有实用性的。

## 系统界面

新的系统界面给我的第一感受就是变圆了，到处都变得更圆了。

这点变化不知道是否是由新设计引入的，但它连鼠标指针都变圆了就很让人难绷……

![新的鼠标指针形状](https://s3-blaze.rqwq.link/vscode/0726129611e355d2aae81d5eaa9afc76.png)

当然指针的默认颜色实际上没有改动，仍然是黑的，截图中的指针是我自己调整的颜色。

### 液态玻璃（Liquid Glass）

本次更新最重磅的更改就是苹果的全新设计风格液态玻璃了，从锁屏界面开始到各种各样的组件，整个系统各处都被换成了液态玻璃的材质与设计。

![整体界面](https://s3-blaze.rqwq.link/vscode/986b0552309370cf22f2b46e12960dd8.png)

之前对这个新设计也没有很反感，反而觉得它做好了其实挺好看的，就是有点吃配置，而且确实在一定程度上影响部分 UI 元素的可读性。但实测下来，现在的液态玻璃性能已经不算差了，也并不会对待机时长造成什么影响，我的体验就是感受不到跟更新前有什么区别。

像 iOS 26 一样，macOS 26 也在设置中提供了一个选项用于切换液态玻璃的两种不同模式，一种是默认的高透效果，而另一种为了照顾可读性牺牲一些视觉效果，为玻璃区域添加背景色。

![液态玻璃选项](https://s3-blaze.rqwq.link/vscode/650f03283494b8b9dcfe45132c997865.png)

### 窗口动画

现在窗口打开和关闭在大部分情况下会有一个变形动画，感觉还是很灵动的，这可能算是个巨大的改进，因为苹果在过去很久都没给窗口加过任何动画。

{% media video %}
- url: https://s3-blaze.rqwq.link/vscode/6b23547bb6a02ce30cd82fe4e349d5b7.webm
{% endmedia %}

至于为什么说是大部分情况，有些时候这个动画会被无端吃掉，原因未知。

### 亮度/音量控件

这两个小控件现在被移到了屏幕上边缘，而且由原先的清晰刻度变成了一条没有直接刻度的线。

### 小组件（Widget）

现在 Widgets 可以被直接放到桌面上了，因此喜欢挂各种参数监控但受限于顶栏空间的友友们现在可以发挥创意装饰自己没用到的桌面空间了。但这改进对我无感，我还是喜欢把这些组件放在通知中心，因为我的桌面时常会被各种各样的文件塞满。

### 顶栏

顶栏现在被改成了透明的，当然也可以在设置中改回原来的样子，但我觉得透明的可能更好看一些。

具体效果可以参考 [液态玻璃](#液态玻璃) 一节的截图。
