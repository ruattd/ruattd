> Walk softly through the world.

这个博客是我一时兴起搭的，本来的目的就是满足自己随手写点什么的需求，但是搭了都搭了我寻思就认真写吧……但是我这人有个特色，就是写东西很认真，但很慢，所以这个博客内容更新可能并不会很多，但是我能保证你看到的一定不会有低质量内容，也一定是我自己手写的（在这个 AIGC 内容泛滥的时代这种声明还是很有重量的，笑）。

之前就看到 Astro 这个框架，觉得它挺万能的，自由程度也比较高，于是就打算用了。恰逢一前端大佬群友 Cosine 发了她自己做的一个主题的 release，偶然在 GitHub 时间线里看到，就拿来用了。然后因为我比较懒，这个网页跟 Cosine 的博客风格几乎一模一样，连图片都没换（跑）。但是我坚信这种网站的重点不在风格，而在内容，持续产出的优质内容才是博客的活力和价值所在——所以我一定会努力写出更好的文章的！

以下是个人简介，与 GitHub 主页 README 内容随机同步（看心情），如果你发现有什么地方不同步请敲我提醒一下。
